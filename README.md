# Videography & Unreal Rendering Portfolio

A professional portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- Clean, minimal dark theme design
- Responsive layout for all devices
- Project showcase with individual project pages
- About and Contact pages
- Optimized for performance
- Ready to deploy to Vercel

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

### Add Your Projects

Edit `data/projects.ts` to add your own projects. Replace the placeholder video URLs with your actual YouTube or Vimeo embeds.

### Update Content

- **Homepage**: Edit `app/page.tsx`
- **About Page**: Edit `app/about/page.tsx`
- **Contact Info**: Edit `app/contact/page.tsx`
- **Site Title**: Edit `app/layout.tsx`

### Add Your Demo Reel

Replace the placeholder in `app/page.tsx` with your actual demo reel embed code.

## Deploy to Vercel

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repository
4. Deploy (it's automatic!)

Your site will be live with a free `.vercel.app` domain. You can add a custom domain in Vercel settings.

## Video Hosting

For best results, host your videos on:
- **YouTube**: Free, unlimited bandwidth
- **Vimeo**: More professional look, 60GB/year free tier
